# Projeto-ISC
Jogo inspirado em Pac-Man, desenvolvido em Assembly RISC-V

Esse projeto foi criado usando “Pac Man” como inspiração para sua jogabilidade, 
desenvolvido em Assembly RISC-V, utilizando, mais especificamente, o RARS como
ambiente de desenvolvimento. O jogo foi desenvolvido como projeto final da matéria
de Introdução a Sistemas Computacionais.
